# Text

Just a common text component to be used for paragpraphs and
all inline html text tags as well.

## HTML valid

A text component which handles HTML validity of text tags.

## Examples

```jsx
import { Text } from '@vitus-labs/elements'

<Text>
  This is a Text <Text>This is an inline text inside paragraph</Text>
</Text>
```

```jsx
import { Text } from '@vitus-labs/elements'

<Text deleted>
  This is a Text <Text replaced>This is an inline text inside paragraph</Text>{' '}
  <Text>Another inline text should be rendered as span</Text>
</Text>
```

## Properties

Below are default propertes.

| Property Name | Type    | Default Value | Description                                                        |
| ------------- | ------- | ------------- | ------------------------------------------------------------------ |
| tag           | any     |               | HTML valid tag                                                     |
| children      |         |               |                                                                    |
| inline        | boolean | false         | Makes element behave like an inline text element (adds inline tag) |

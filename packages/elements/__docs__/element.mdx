# Element

Element is the smallest possible unit to be used for creating components.
By default it renders `div`. You can render any valid HTML `tag` by passsing
it as a prop.

Element is by default _inline-flex_. You can set _flex_ by using boolean `block`
value which will also stretch the element to width of _100%_.

Element also checks by default if `tag` passed in is type of `inline` HTML tag
like `button` or `span`. If so, other helper sub-elements rendered
inside (`beforeContent` and `afterContent` wrappers) are by default converted
to `span` element so it is HTML valid.

## Children

To render children inside component you can pass content as `children` prop
or `label` prop which might be nicer or more comfortable for inline elements.
If you pass both of them _children_ **will have priority** to be rendered
and _label_ will be ignored.

## Before & After content

You can also pass `beforeContent` or `afterContent`. This is kind of alternative
to using `:before` or `:after` pseudo elements. This is useful especially in cases
like buttons with loading icon or input wrppers with icons or anything similar.

## Vertical vs. horizontal alignment

Element can be aligned horizontally or vertically. So you can have `beforeContent`
on the left side of `children` or above. The same applies for `afterContent`
(below children or on the right side). Check out the examples below to see more.

For vertical aligning you can use prop `vertical` set to `true`.

## Aligning sub-elements inside

Each sub-element has its own options to align itself. Each element has options to set
_direction_, _horizontal_ and _vertical_ alignment by using props `contentDirection`,
`contentAlignX` or `contentAlignY`. For pseudo elements `beforeContent` and `afterContent`
you can use the same props but changed prefix according to its name
(`beforeContentDirection` etc.) See the examples below.

You don't need to write flex css styles and think about proper styling when changing
direction anymore.

## Extending styles

It is also possible to extend default styles of `content`, `beforeContent`
and `afterContent` elements just by passing props `contentCss`, `beforeContentCss`
and `afterContentCss`. To style root Wrapper, you can use `css` prop.

You can pass simple string:

```jsx
<Element contentCss=`text-align: center;` />
```

or styles defined inside `css` function from styled-components

```jsx
<Element
  contentCss={css`
    text-align: center;
  `}
/>
```

or just pass function, because each prop returns `css` as a parameter,
so you don't need to import it by yourself (and safe one line of code):

```jsx
<Element
  contentCss={(css) =>
    css`
      text-align: center;
    `
  }
/>
```

The same rules applies to all `css` properties.

## Properties

Below are default propertes.

| Property Name          | Type                                                  | Default Value | Description                                                                            |
| ---------------------- | ----------------------------------------------------- | ------------- | -------------------------------------------------------------------------------------- |
| tag                    | HTML valid tag                                        |               | HTML valid tag                                                                         |
| innerRef               | React `ref` name, can be used as an alternative to it |               | React `ref`                                                                            |
| label                  | `ReactNode`                                           | -             | Content to be rendered, priorities `children` => `contnent` => `label`                 |
| children               | `ReactNode`                                           | -             | Content to be rendered, priorities `children` => `contnent` => `label`                 |
| content                | `ReactNode`                                           | -             | Content to be rendered, priorities `children` => `contnent` => `label`                 |
| block                  | boolean                                               | false         | Makes element behave like block element and inside `flex` element make it `100% width` |
| equalCols              | boolean                                               | false         | Makes all three boxes of equal size                                                    |
| gap                    | number                                                |               | Margin between elements before - content - after                                       |
| beforeContent          | `ReactNode`                                           | -             | A content to be rendered before main content                                           |
| afterContent           | `ReactNode`                                           | -             | A content to be rendered after main content                                            |
| vertical               | boolean                                               | false         | Makes elements inside vertically aligned                                               |
| alignX                 | `left`,`center`,`right`,`spaceBetween`,`spaceAround`  |               | alignment on axis X of elements inside _wrapper_ content component                     |
| alignY                 | `top`,`center`,`bottom`,`spaceBetween`,`spaceAround`  |               | alignment on axis Y of elements inside _wrapper_ content component                     |
| css                    | `string`, `func`                                      |               | css styles to be passed to _wrapper_ element component                                 |
| contentCss             | `string`, `func`                                      |               | css styles to be passed to _main_ content component                                    |
| beforeContentCss       | `string`, `func`                                      |               | css styles to be passed to _before_ content component                                  |
| afterContentCss        | `string`, `func`                                      |               | css styles to be passed to _after_ content component                                   |
| contentDirection       | `inline`,`rows`,`reverseRows`,`reverseInline`         | `inline`      | alignment of elements inside _main_ content component                                  |
| contentAlignX          | `left`,`center`,`right`,`spaceBetween`,`spaceAround`  | `left`        | alignment on axis X of elements inside _main_ content component                        |
| contentAlignY          | `top`,`center`,`bottom`,`spaceBetween`,`spaceAround`  | `center`      | alignment on axis Y of elements inside _main_ content component                        |
| beforeContentDirection | `inline`,`rows`,`reverseRows`,`reverseInline`         | `inline`      | alignment of elements inside _before_ content component                                |
| beforeContentAlignX    | `left`,`center`,`right`,`spaceBetween`,`spaceAround`  | `left`        | alignment on axis X of elements inside _before_ content component                      |
| beforeContentAlignY    | `top`,`center`,`bottom`,`spaceBetween`,`spaceAround`  | `center`      | alignment on axis Y of elements inside _before_ content component                      |
| afterContentDirection  | `inline`,`rows`,`reverseRows`,`reverseInline`         | `inline`      | alignment of elements inside _after_ content component                                 |
| afterContentAlignX     | `left`,`center`,`right`,`spaceBetween`,`spaceAround`  | `right`       | alignment on axis X of elements inside _after_ content component                       |
| afterContentAlignY     | `top`,`center`,`bottom`,`spaceBetween`,`spaceAround`  | `center`      | alignment on axis Y of elements inside _after_ content component                       |

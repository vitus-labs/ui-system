# Custom count of columns

By default, we have defined columns to be 12. You can easily change
it in your theme provider or pass it as `columns` prop to `Container`
or `Row`.

## 12 columns grid

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;

    >
      <Row>

## 7 columns grid via Row

The first row has 12 columns and the second one has 7 columns.
So even in one Container, you can have different numbers
of columns.

# Container

## Usage example

Use default grid settings by importing it to styled-components
`<ThemeProvider>`

```jsx
import { ThemeProvider } from 'styled-components'
import { Container, Row, Col, theme } from 'coolgrid'

<ThemeProvider theme={theme}>
  <Container>
    <Row>...</Row>
    <Row>...</Row>
  </Container>
</ThemeProvider>
```

or create your own grid configuration. There are several approaches
how you can achieve this.

```jsx
import { ThemeProvider } from 'styled-components'
import { theme } from 'coolgrid'

<ThemeProvider theme={theme}>{/* ...your components */}</ThemeProvider>
```

## Properties

| property name | type                     | default value | description                                                                    |
| ------------- | ------------------------ | ------------- | ------------------------------------------------------------------------------ |
| theme         | object                   |               | `styled-components` theme                                                      |
| breakpoints   | object                   |               | breakpoints for responsivity                                                   |
| rootSize      | number                   | 16            | base font size used for `em` calculation                                       |
| columns       | number                   | 12            | total number of columns in one `Row`                                           |
| size          | number / object          |               | size of `Col`s inside `Container`                                              |
| gap           | number / object          | -             | gap of all `Col`s inside `Container`                                           |
| padding       | number / object          | -             | padding of all `Col`s inside `Container`                                       |
| gutter        | number / object          | -             | gutter of all `Col`s inside `Container`                                        |
| css           | string / func            | -             | css extension for `Container`                                                  |
| colCss        | string / func            | -             | css extension for all `Col`s inside `Container`                                |
| rowCss        | string / func            | -             | css extension for all `Row`s inside `Container`                                |
| component     | string / react component |               | valid HTML tag / component, generates `div` by default                         |
| colComponent  | string / react component | -             | valid HTML tag / component for all `Col`s inside `Container`, `div` by default |
| rowComponent  | string / react component | -             | valid HTML tag / component for all `Row`s inside `Container`, `div` by default |

You can also use `breakpoint keys` as a properties. The example would be following:

```jsx
  const xs = {
    padding: 10,
    gap: 10,
    gutter: 0,
    size: 4
  }

  const lg = {
    padding: 10,
    gap: 10,
    gutter: 0,
    size: 4
  }

  <Container xs={xs} lg={lg} />
```

From the example above, data are passed via context to Row and Col components.
Component `Row` takes the value of `gap` and `gutter`.
Component `Col` gets values of `size`, `gap` and `padding`.

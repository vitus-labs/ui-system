# Unequal columns

## Using size prop in Col

Column width can be set for all breakpoints by using `size` property.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col size={2}>size 2</Col>
        <Col size={3}>size 3</Col>
        <Col size={7}>size 7</Col>
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Size via viewport keys in Col

Default viewports come from Bootstrap where are predefined
breakpoints for `xs`, `sm`, `md`, `lg`, `xl` screens.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col xs={2} sm={2} md={2} lg={2} xl={2} />
        <Col xs={3} sm={3} md={3} lg={3} xl={3} />
        <Col xs={7} sm={7} md={7} lg={7} xl={7} />
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Size via viewport keys in Row

Default viewports come from Bootstrap where are predefined
breakpoints for `xs`, `sm`, `md`, `lg`, `xl` screens. You
can define them on Row and it will be passed to `Col`s
via context.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row xs={4} sm={4} md={4} lg={4} xl={4}>
        <Col>size 4</Col>
        <Col>size 4</Col>
        <Col>size 4</Col>
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

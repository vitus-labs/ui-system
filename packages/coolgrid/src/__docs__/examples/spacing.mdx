## Define gap on Container

You can define `gap` on `Container` component for all Rows and columns
inside Container. But if you need, you can easily override it within
each Row component.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      gap={10}
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
      </Row>
      <Row gap={0}>
        <Col size={6}>gap-0</Col>
        <Col size={6}>gap-0</Col>
        <Col size={6}>gap-0</Col>
        <Col size={6}>gap-0</Col>
      </Row>
      <Row>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
        <Col size={6}>gap-10</Col>
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

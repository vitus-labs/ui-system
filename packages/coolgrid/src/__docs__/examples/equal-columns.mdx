# Equal columns

## Example

The default behavior is that all `Col`s in `Row` have the same width.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col />
        <Col />
        <Col />
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Using size prop in Col

Column width can be set for all breakpoints by using `size` property.
For example `size={4}` creates column of width `33.33333%` when `columns`
parameter is set to `12`.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col size={4}>size 4</Col>
        <Col size={4}>size 4</Col>
        <Col size={4}>size 4</Col>
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Define size prop in Row

You can define `size` prop in Row and it will be passed to all
`Col`s via context. So you can write less to achieve the same
result.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row size={4}>
        <Col>size 4</Col>
        <Col>size 4</Col>
        <Col>size 4</Col>
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Size via viewport keys in Col

Default viewports come from Bootstrap where are predefined
breakpoints for `xs`, `sm`, `md`, `lg`, `xl` screens.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col xs={4} sm={4} md={4} lg={4} xl={4} />
        <Col xs={4} sm={4} md={4} lg={4} xl={4} />
        <Col xs={4} sm={4} md={4} lg={4} xl={4} />
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Size via viewport keys in Row

Default viewports come from Bootstrap where are predefined
breakpoints for `xs`, `sm`, `md`, `lg`, `xl` screens. You
can define them on Row and it will be passed to `Col`s
via context.

        <Col>size 4</Col>

      </Row>
    </Container>
  </ThemeProvider>

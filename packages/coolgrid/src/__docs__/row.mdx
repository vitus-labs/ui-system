# Row

## Usage example

Use default grid settings by importing it to styled-components
`<ThemeProvider>`

```jsx
import { ThemeProvider } from 'styled-components'
import { Container, Row, Col, theme } from 'coolgrid'

<ThemeProvider theme={theme}>
  <Container>
    <Row>...</Row>
    <Row>...</Row>
  </Container>
</ThemeProvider>

// or without using Container
<ThemeProvider theme={theme}>
  <Row>...</Row>
  <Row>...</Row>
</ThemeProvider>
```

or create your own grid configuration. There are several approaches
how you can achieve this.

```jsx
import { ThemeProvider } from 'styled-components'
import { theme } from 'coolgrid'

<ThemeProvider theme={theme}>{/* ...your components */}</ThemeProvider>
```

you can also directly overrides grid settings in your Row components

```jsx
import { ThemeProvider } from 'styled-components'
import { Row, theme } from 'coolgrid'

<ThemeProvider theme={theme}>
  <Row columns={7}>{/* ...your components */}</Row>
</ThemeProvider>
```

## Properties

| property name | type                     | default value | description                                                              |
| ------------- | ------------------------ | ------------- | ------------------------------------------------------------------------ |
| theme         | object                   |               | `styled-components` theme                                                |
| breakpoints   | object                   |               | breakpoints for responsivity                                             |
| rootSize      | number                   | 16            | base font size used for `em` calculation                                 |
| columns       | number                   | 12            | total number of columns in one `Row`                                     |
| size          | number / object          |               | size of `Col`s inside `Row`                                              |
| gap           | number / object          | -             | gap of all `Col`s inside `Row`                                           |
| padding       | number / object          | -             | padding of all `Col`s inside `Row`                                       |
| gutter        | number / object          | -             | gutter of all `Col`s inside `Row`                                        |
| css           | string / func            | -             | css extension for `Row`                                                  |
| colCss        | string / func            | -             | css extension for all `Col`s inside `Row`                                |
| component     | string / react component |               | HTML valid tag / component, generates `div` by default                   |
| colComponent  | string / react component | -             | valid HTML tag / component for all `Col`s inside `Row`, `div` by default |

You can also use `breakpoint keys` as a properties. The example would be following:

```jsx
  const xs = {
    padding: 10,
    gap: 10,
    gutter: 0,
    size: 4
  }

  const lg = {
    padding: 10,
    gap: 10,
    gutter: 0,
    size: 4
  }

  <Row xs={xs} lg={lg} />
```

From the example above, data are passed via context to Col components.
Component `Col` gets the values of `size`, `gap` and `padding`.

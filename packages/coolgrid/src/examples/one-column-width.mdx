import { Playground } from 'docz'
import { ThemeProvider } from 'styled-components'
import { Container, Row, Col, theme } from '../../src'

# One column width

## Using size prop in Col

We can define `size` only on some columns.
The other columns will fit into the rest width.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col />
        <Col size={5}>size 5</Col>
        <Col />
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

## Size via viewport keys in Col

The example below should look the same as the example above.
Instead of using `size` property, we can also use **viewport keys**
to define column size.

<Playground>
  <ThemeProvider theme={theme}>
    <Container
      colCss={`
        background-color: #e0e0eb;
        border: 1px solid #b3b3cc;
        padding: 20px;
      `}
    >
      <Row>
        <Col />
        <Col xs={5} sm={5} md={5} lg={5} xl={5} />
        <Col />
      </Row>
    </Container>
  </ThemeProvider>
</Playground>

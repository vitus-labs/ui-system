# Handling visual states

## Component theme

By default each component accepts default theme provided by ThemeProvider.
But it's helpful to define customized theme for any component.

Let's assume we have a component defined like this:

```
import rocketstyle from 'rocketstyle'
import Element from './someReactComponent'

export default rocketstyle({
  name: 'Element.Name',
  component: Element
})
  .styles(
    css => css`
      ${({ theme: t }) => css`
        line-height: ${t.lineHeight};
        font-family: ${t.fontFamily};
        color: ${t.color};
      `};
    `
  )
```

The component above will render defined css properties with values
from root theme. If you have already created your rocketstyle
component, you can easily override theme by using:

```
  import element from './previousExample'

  element.theme({
    lineHeight: 2,
    fontFamily: 'Arial',
    color: black;
  })

  // you can also access theme and based on that create your own theme
  element.theme(theme => ({
    lineHeight: theme.lineHeight * 2,
    fontFamily: 'Arial',
    color: theme.colors.primary;
  }))

```

## Different visual states

Rocketstyle is helpful tool when you need to define different
visual states of your components. Below is the list of supported
functions, which helps you split the logic into easily understandable
pieces which forces you to define your states properly:

- states
- sizes
- variants
- multiple

Each of the following functions expect to have as an argument plain object:

```jsx
element.states({
  primary: {
    bgColor: '#fff'
  },
  secondary: {
    bgColor: '#000'
  }
})
```

or a function with arguments `theme` (from the root ThemeProvider):

```jsx
element.sizes(theme => ({
  small: {
    fontSize: theme.fontSize.small
  },
  large: {
    fontSize: theme.fontSize.large * 2
  }
}))
```

## Define colorful states

States are mean to be somehow visually different in terms of colors (mostly).
But feel free to define the meaning by yourself if it fits your needs.

Let's create a button with `primary` and `secondary` state.

```jsx
import rocketstyle from 'rocketstyle'

const Button = rocketstyle({
  name: 'Element.Name',
  component: 'button'
})
  .styles(
    // define button css styles and pass values from theme
    css => css`
      ${({ theme: t }) => css`
        background-color: ${t.bgColor};
        color: ${t.textColor};
      `};
    `
  )
  .states(t => ({
    // define different button visual states
    primary: {
      bgColor: 'papayawhip',
      textColor: 'palevioletred'
    },
    secondary: {
      bgColor: 'black',
      textColor: 'white'
    }
  }))
```

Now it's time to see it in practice!

```jsx
// now we can use out Button!
<Button state="primary">This is a primary button!</Button>

// or by passing boolean prop, isn't it cool?
<Button primary>This is a primary button!</Button>
```

## Define sizes

It works exactly the same way as `states`. Let's create a button
with `small` and `large` size.

```jsx
import rocketstyle from 'rocketstyle'

const Button = rocketstyle({
  name: 'Element.Name',
  component: 'button'
})
  .styles(
    // define button css styles and pass values from theme
    css => css`
      ${({ theme: t }) => css`
        height: ${t.height};
        font-size: ${t.fontSize};
      `};
    `
  )
  .sizes(t => ({
    // define different button visual sizes
    small: {
      height: '20px',
      fontSize: '12px'
    },
    large: {
      bgColor: '40px',
      fontSize: '30px'
    }
  }))
```

Now it's time to see it in practice!

```jsx
// now we can use out Button!
<Button size="small">This is a small button!</Button>

// or by passing boolean prop
<Button small>This is a small button!</Button>
```

## Define variants

It works exactly the same way as `states`. Let's create a button
with `small` and `large` size. They are meant to be something specific
to your component. For example, you want `top` or `bottom` menu.
By default the visual looks the same, there are only few differences
in styling for each state.

```jsx
import rocketstyle from 'rocketstyle'

const Menu = rocketstyle({
  name: 'Element.Name',
  component: 'button'
})
  .styles(
    // define button css styles and pass values from theme
    css => css`
      ${({ theme: t }) => css`
        height: ${t.height};
        font-size: ${t.fontSize};
      `};
    `
  )
  .variants(t => ({
    top: {
      // properties
    },
    bottom: {
      // properties
    }
  }))
```

Now it's time to see it in practice!

```jsx
<Menu variant="top" />

//or using boolean
<Menu top />
```

## Define multiple

Multiple works a little bit differently. They are type of `array` of defined
states. In other words, you can combine more multiple variants together.
It can be useful in cases like styling text. You may want to have a Text
component which can be _underlined_ and _bold_ at the same time.

```jsx
import rocketstyle from 'rocketstyle'

const Text = rocketstyle({
  name: 'Element.Name',
  component: 'p'
}).multiple({
  underline: {
    // something
  },
  bold: {
    // something
  }
})
```

Now it's time to see it in practice!

```jsx
// now we can use out Text!
<Text multiple={['underline', 'bold']}>Text</Text>

// or by passing boolean prop, isn't it cool?
<Text underline bold>Text</Text>
```

## Put it all together

Here is one a little bit more complex example where you can see how
simple is that! In few minutes we have a `button` with states(primary,
secondary), sizes (small, large) and differentiators (rounded, outlined).

The below component allow us using `Button` component as following:

```jsx
<Button primary>Primary button</Button>
<Button primary small>Primary small button</Button>
<Button large rounded outlined>Large rounded outlined button</Button>

// or alternatively
<Button state="primary">Primary button</Button>
<Button state="primary" size="small">Primary small button</Button>
<Button state="large" multiple={['rounded', 'outlined']}>Large rounded outlined button</Button>
```

```jsx
import rocketstyle from 'rocketstyle'

const Button = rocketstyle({
  name: 'Element.Name',
  component: 'button'
})
  .styles(
    // define button css styles and pass values from theme
    css => css`
      ${({ outline, theme: t }) => css`
        height: ${t.height};
        font-size: ${t.fontSize};
        background-color: ${outline ? 'transparent' : t.bgColor};
        color: ${outline ? t.bgColor : t.textColor};
        border-radius: ${t.borderRadius};
      `};
    `
  )
  .states({
    primary: {
      bgColor: 'papayawhip',
      textColor: 'palevioletred'
    },
    secondary: {
      bgColor: 'black',
      textColor: 'white'
    }
  })
  .sizes({
    small: {
      height: '20px',
      fontSize: '12px'
    },
    large: {
      bgColor: '40px',
      fontSize: '30px'
    }
  })
  .multiple({
    outline: true,
    rounded: {
      borderRadius: '8px'
    }
  })
```
